## Highlights

### Versions (opt-in)
Settings → Versions turns on local Git history for the active vault. Save version / History / Restore from a note; Advanced Git covers worktrees and diffs. Off by default. Windows can install portable MinGit into the data folder.

### Virtual desktops
Up to four named desktop layouts. Status-bar strip shows occupied slots; `Ctrl+; 1–4` switches; Spotlight keeps create / rename / switch. Persistence migrates to shell-tabs v3 (older layouts become one “Main” desktop).

### Status bar & rails
Overflow (`…`) is gone. Automations shows `enabled/total`. Activity pulse and contextual whispers stay quiet. Rails gain popovers, shake-to-reveal / `Ctrl/Cmd+Shift+.` summon, and per-surface toolbars. Titlebar adds New Tab and per-tab back/forward.

### Liquid & vault
Snapshot timelines, ` ```feed``` ` for Stasis last-good, Trash restore UI, note create-only (no silent overwrite), property/tag controls, and Scripts CodeEditorShell (Grapheme-first).

### Chat & Work
Presence empty state for new chats; Work surface hosts Asks.

## Upgrade notes

- **Shell layout:** localStorage key moves to `medousa-home-shell-tabs-v3`. Prior v1/v2 layouts migrate into a single “Main” desktop.
- **Hotkeys:** `Ctrl+; 1–4` switches virtual desktops (max 4). Pane-by-index focus via those digits is gone.
- **Status bar:** overflow menu removed; look for the desktop strip and Automations `r/n` control.
- **Versions:** remains off until enabled; enabling may install Git (Windows) or prompt for CLT / package install.
- **Note create:** colliding title/path is refused instead of overwriting.
- **Stasis:** engine depends on Stasis **0.8.0** — validate feeds / Grapheme paths after upgrade.
- This cut ships **engine + desktop** only (`ship_engine` + `ship_desktop`). Adapters, installer, mcp-gateway, and local-brain stay on **0.4.1** channel artifacts. Do **not** push a `v0.5.0` full-train tag unless every package stamp is raised together.

# Medousa 0.4.1

**Live that folds, papers that fit, and loose files that stay alive.**

## Highlights

### Loose files behave like notes in Live
Absolute markdown paths stash and restore buffers (and editor UI) so LME keep-alive / multi-pane hosts no longer mount an empty TipTap. Saves still write to the absolute path. Vault graph features (wikilinks / backlinks) stay empty for true orphans.

### Callouts, CSV, and more languages
Preview callout icons line up with titles. Copy CSV no longer opens Configure. Fenced code highlight adds C / C++ / C# / Java / PHP / R / Scala (go/sql already shipped).

### Fold, paper, hide marks, size embeds
Heading and list chevrons fold sections like Obsidian. Overflow menu: hide markdown marks, cycle paper width. Compare / slides get a width control (`width:` in the fence); tables get a Live width cycle + collapse chrome.

### Sheet / workbook / slides stay object-first
Kind pill no longer snaps back to Note after tab switches. Switching to Sheet seeds a grid and opens table view (View/Raw toggle). Workbook markers get a manifest surface instead of blank Live. Slides keep deck mode the same way boards already did.

---

## Upgrade notes

- No migration. Existing notes keep working.
- `width:` on compare/slides is optional; omit for default wide.
- Fold state is session-local (not written into markdown).
- Wikilinks / vault views that need a vault root remain degraded for files outside any vault.
- Promoting a note to Sheet/Workbook may seed a starter table or marker frontmatter on first switch.

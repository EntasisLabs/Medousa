## Highlights

### Styled blocks + block ids
Liquid ` ```block ` fences become typography containers (font, size, align, spacing). Trailing Obsidian-style `^block-id` markers round-trip on paragraphs and headings.

### Live format bubble, progressive disclosure
Shape page for marks, block type, lists, and link; Voice page for typeface and color (Build-style wheel + hex). Styled-block chrome uses Type / Layout doors instead of chip landfill.

### Content zoom without breaking Live
`⌘`/`Ctrl` `+` / `-` / `0` scales notes, chats, and scripts via content font tokens — not CSS `zoom` — so typing and format actions stay scroll-stable.

### Footnotes, callouts, code highlight
Markdown footnotes in Live and Preview, refreshed callout styling, and syntax highlighting for fenced code.

### Daemon agents + ACP
Home can talk to external agent runtimes through the daemon agents surface and ACP client wiring. Workbooks foundation and a stronger slides player land alongside.

---

## Upgrade notes

- No migration. Existing vault notes and liquid fences keep working.
- New `block` fences are opt-in via slash / templates; older notes are unchanged.
- Content zoom is per-shell session UI scale for content surfaces; it does not change pane layout maximize.

## Highlights

### Shell tabs & split panes
Open Workspace, Chat, and Web as tabs. Split a pane when you need two surfaces at once. Chat sessions can stay live in more than one pane; notes keep a per-pane buffer so background editors don’t stomp the focused note.

### Quieter composer
Attach, Profile, and Agent live under `+`. Non-default profile/agent show as dismissible chips. Manage links land on Profiles / Agents (desktop shell tabs or mobile More).

### Inline images in notes
Paste or drop an image into Live or Build — it embeds as `![…](data:image/…)` and renders in Preview. Soft 2 MB cap; larger files still belong in Linked files.

### Liquid slash catalog
Type `/` in a note for starters covering the full Liquid fence set: card, carousel, actions, section, chips, media, cite, compare, plan, timeline, shortlist, decision, brief, chart, dashboard, report, slides, tabs, steps, accordion, code, tree, mini board — plus writing helpers and kanban / query view.

### Errors you can expand
Failed turns keep the friendly line and, when the daemon sends a distinct debug message, a collapsed **View details** on the error callout.

### Onboarding models
Provider pick in the welcome flow uses the same catalog → live → default resolution as Settings.

---

## Upgrade notes

- Existing environment presets that still point Workspace at `book-open` keep working; new defaults use `panels-top-left`. Notes / vault chrome still use the book icon.
- Notes that already contain escaped paste leftovers (`!\[…](data:…)`) need a fresh paste — they won’t auto-heal.
